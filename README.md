# MERN E-Commerce Website

## Description
A full-stack E-Commerce application built using the MERN Stack with secure authentication, product management, shopping cart, and role-based access control.

## Features
- User Authentication (JWT)
- Product Search
- Shopping Cart
- Admin Dashboard
- CRUD Operations
- Responsive Design

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- Axios
- JWT
- Context API
